About This Workshop

Level up your LLM skills — from basics to agents

Prompting is no longer just "talking to ChatGPT." It's a core engineering skill that separates average AI developers from exceptional ones. This masterclass teaches you to think like the model — understanding context windows, token prediction, and reasoning chains.

In three focused hours, you'll move from fundamentals to building a functional AI mini-agent that can use tools, reason step-by-step, and handle multi-turn conversations. We'll work live with GPT-4o and Gemini 1.5 Pro APIs.

Session Curriculum

What you'll cover

1
10:00 – 10:45 AM
LLM Foundations & Setup
Understand how LLMs predict tokens, what temperature and top-p control, and set up your API keys for GPT-4o and Gemini. Overview of the prompt lifecycle: system → user → assistant.
OpenAI SDKGemini APIPython
2
10:45 – 11:30 AM
Core Prompting Strategies
Zero-shot vs. few-shot prompting. Role prompting and persona injection. Output formatting with structured JSON. Chain-of-thought (CoT) reasoning and self-consistency. Practice exercises after each technique.
Zero-ShotFew-ShotCoTStructured Output
3
11:30 AM – 12:00 PM
Advanced Patterns: ReAct & Tool Use
The ReAct (Reason + Act) framework. Function calling with OpenAI's tool_calls format. Giving your model access to a calculator, a web search stub, and a file reader. Handling multi-turn agent loops.
ReActFunction CallingTool Use
4
12:00 – 1:15 PM
Hands-On: Build Your Mini-Agent
Guided project: build a personal research agent that can search topics, summarize findings, and write a structured report. You'll add memory, tools, and a retry loop when the model fails. Mentor support available throughout.
AgentsMemoryLangChainPython
5
1:15 – 1:30 PM
Showcase & Q&A
Participants demo their agents live. Open Q&A with the instructor. Resources, next steps, and community challenge announcement.
DemoQ&A
Prerequisites

What you should know

  • Basic Python — you should be comfortable writing functions and calling APIs.
  • Familiarity with what LLMs are (ChatGPT, Gemini, etc.) at a user level.
  • A Google or OpenAI account to generate a free API key (instructions will be shared pre-event).
  • VS Code or any Python IDE installed on your machine.
  • No prior prompting engineering experience required — we start from scratch.
